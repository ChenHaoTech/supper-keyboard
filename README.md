# space mode

### space 

- hjkl ==> 鼠标的 左上下右 移动
- 垂直水平移动;
  - i ==> 鼠标垂直到头
  - m ==> 鼠标垂直到底
  - p==> 鼠标水平到左头
  - ;==>鼠标水平到右头
- 短程移动
  - y==>鼠标上移3格
  - b==>鼠标下移3格

# capslock mode

### capslock

- hjkl==>光标的左上下右

- 垂直水平移动
  - p==>光标移动到行首
  - ;==>光标移动到行尾
  - i==>光标移动到文件顶端
  - m==>光标移动到文件末尾
- 短程移动
  - y==>光标上移5行
  - b==>光标下移五行

- 单词跳转
  - u==>单词前跳转
  - o=>单词后跳转

### capslock+shift

- hjkl==>shift+光标的左上下右
- 垂直水平移动
  - p==>shift+光标移动到行首
  - ;==>shift+光标移动到行尾
  - i==>shift+光标移动到文件顶端
  - m==>shift+光标移动到文件末尾
- 短程移动
  - y==>shift+光标上移5行
  - b==>shift+光标下移五行
- 单词跳转
  - u==>shift+单词前跳转
  - o=>shit+单词后跳转

### capslock+win

- 等价于单击capslock

### capslock+backspaec

- 删除一整行
