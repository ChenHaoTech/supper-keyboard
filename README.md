# space mode

### space 

-  hjkl ==> 鼠标的 左上下右 移动
- 垂直水平移动
  - i ==> 鼠标垂直到头
  - m ==> 鼠标垂直到底
  - p==> 鼠标水平到左头
  - ;==>鼠标水平到右头
- 短程移动
  - y==>鼠标上滑3格
  - b==>鼠标下滑3格

### space+alt

​		

- hjkl==>滚轮的 左上下右的移动
- 垂直水平移动
  - i==>到达窗口顶端
  - m=>到达窗口底端
  - p=>到达窗口最左端
  - ;=>到达窗口最右端
- 短程移动
  - y==>滚轮上滑三格
  - b==>滚轮下滑三格

### space+enter

- 光标当前位置左单击

### space+enter

- 光标当前位置右单击

# esc mode

### esc

- hjkl==>光标的左上下右

- 垂直水平移动
  - p==>光标移动到行首
  - ;==>光标移动到行尾
  - i==>光标移动到文件顶端
  - m==>光标移动到文件末尾
- 短程移动
  - y==>光标上移5行
  - b==>光标下移五行

- 单词跳转
  - u==>单词前跳转
  - o=>单词后跳转

### esc+alt

- hijk==>alt键+左上下右

### esc+win

- 等价于单击capslock
